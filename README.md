# Getting Started

1. Clone reposetory `git clone https://github.com/murove1/my-start-gulp`;
2. Install node_modules `npm i`;
4. Run `gulp`.

# Gulp task

1. <b>gulp</b> - run gulp task (sass, js, browserSync);
2. <b>build</b> - build project to dist folder.
